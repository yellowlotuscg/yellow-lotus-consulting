Preferred verification methods:

1. HTML file upload verification
   - Google provides a file like `google123abc.html`.
   - Put that file at the website root.
   - Deploy to Vercel.
   - Click Verify.

2. DNS TXT verification
   - Google provides a TXT value.
   - Add it in the domain DNS provider.
   - Wait for propagation.
   - Click Verify.
